Find the top-most card

The system's true entry point. Usually an API gateway, a queue consumer, or a scheduled job runner.

Trace one path downward

Follow a single trace from top card to bottom. Pick one and finish it before starting another.

Note every fork

A card that fans out to multiple downstream cards is a decision point. List them as you go.

Look for the missing card

An experienced reader spots what is not drawn. A worker layer with no queue feeding it tells you a queue exists somewhere.
