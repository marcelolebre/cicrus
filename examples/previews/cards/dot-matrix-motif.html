Dots only — never as a container border, never as a button style. The grid is uniform 12–16px, dots 0.5–2px, opacity 0.1–0.2 for backgrounds and full-strength only for data. The motif is the residue of the dot-matrix typography — the same vocabulary, scaled down. Reach for it sparingly: hero backings, empty states, the occasional data-viz floor.
