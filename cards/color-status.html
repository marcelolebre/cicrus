Status & accent
COLOR ENCODES STATE — NOT DECORATION
ACCENT
--accent / --error #D71921 Signal light. Destructive, errors, failed states. One per screen, never decorative.
SUCCESS
--success #4A9E5C Confirmed, completed, connected. UP services. Step counts.
WARNING
--warning #D4A843 Caution, pending, degraded. Triage states. Monitor badges.
INTERACTIVE
--interactive #5B9BF6 · #007AFF light Tappable text. Links, picker values, resource badges. Not for buttons.
DOCTRINE
Monochrome is the canvas. Color is an event. Apply color to the value, not the row background. Labels stay --text-secondary; values flip to status.
RULES
One accent moment per screen, or zero.
Differentiate by opacity or pattern before color.
Badges color the text + border only — never the fill.
Status dot pulses on down, static on up. No checkmarks.
