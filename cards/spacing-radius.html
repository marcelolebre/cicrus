Spacing as meaning
TIGHT 4–8 "These belong together." Icon + label, number + unit.
MEDIUM 16 "Same group, different items." List rows, form fields.
WIDE 32–48 "New group starts here." Section breaks.
VAST 64–96 "This is a new context." Hero → content, major divisions.
Radii — never exceed 16px on cards
--radius-tech 4px Alert banners, modals, inline-detail surfaces. Anything that should read as a technical readout.
--radius-card 8px Task / service / stat cards, kanban columns. The default card corner.
--radius-pill 999px Buttons, badges, toggle tracks, theme bar. Any pill-shape interactive surface.
